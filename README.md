# Aplicaciones-hibridas-tp2
## Inicialización
Para inicializar la base de datos ejecutar: 
``` 
cd backend
npm run init
```
El init crea 3 eventos de prueba y 2 usuarios, uno de ellos es admin.

## Ejecución
Para ejecutar Node
```
cd backend
npm run server
```
Para ejecutar front
```
cd frontend
npm start
```
